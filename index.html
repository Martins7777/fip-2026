🟢 Sincronizacion con Google Sheets

Los datos se guardaran en tu Google Sheet via Apps Script. Cualquier persona con la URL podra leer y escribir los mismos datos.

Pasos para configurar (una sola vez):
1. Abre script.google.com y crea un nuevo proyecto
2. Pega el codigo Apps Script que aparece abajo y guarda
3. Click en Implementar → Nueva implementacion → Aplicacion web
4. Ejecutar como: Yo | Acceso: Cualquier persona
5. Copia la URL de implementacion y pegala abajo
function doGet(e){ var ss=SpreadsheetApp.getActiveSpreadsheet(); var sh=ss.getSheetByName('data')||ss.insertSheet('data'); var val=sh.getRange('A1').getValue(); var res=ContentService.createTextOutput(val||'{}'); res.setMimeType(ContentService.MimeType.JSON); return res; } function doPost(e){ var ss=SpreadsheetApp.getActiveSpreadsheet(); var sh=ss.getSheetByName('data')||ss.insertSheet('data'); sh.getRange('A1').setValue(e.postData.contents); return ContentService.createTextOutput('ok'); }
